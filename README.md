# workflows

记录一些用到的 github actions 的 workflows 。

workflows 文件放在项目目录的 `.github/workflows` 下。

- [ci](https://github.com/LoTwT/workflows/tree/main/ci) ：常规项目检查
- [deploy-ecs](https://github.com/LoTwT/workflows/tree/main/deploy-ecs) ：使用 github actions 自动部署项目到云服务器
- [github-release](https://github.com/LoTwT/workflows/tree/main/github-release) ：使用 github actions 创建 github release
